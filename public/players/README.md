## Сайн үр дүн гаргах зөвлөмж

- Царай нь тод, ойрхон харагдсан, чанартай зураг сонгоорой
- Аль болох урд талаас нь харсан зураг байвал сайн
- Багийн өмсгөлтэй зураг байвал бүр сайн
- Нэг зурагт зөвхөн нэг хүн байх ёстой
